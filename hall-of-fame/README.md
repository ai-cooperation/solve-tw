## 什麼是「被採用」？

當出題企業（Problem Owner）在 Review 中標記一個解法為 **Adopted**，代表：
- 企業實際使用了這個解法（完整或部分）
- 或解法的思路直接影響了企業的決策

**Adopted 是 SolveTW 最高的榮譽。**
解題者獲得 ×3.0 聲望加成，且永久顯示在 Hall of Fame。

## 如何被採用？

沒有捷徑。做到以下幾點，機率最高：

1. **真正理解問題** — 不是只看到表面需求
2. **解法可以直接用或稍改即用** — 不是只有概念
3. **Decision Log 展現判斷力** — 讓企業知道你懂取捨
4. **主動跟進** — 在 Review 後根據回饋改進
