# Solution Templates

> 被 Fork 3 次以上的優質解法，整理為可複用的模板。

## 什麼是 Solution Template？

當一個解法被多次複用，代表它有通用價值。
原作者可以將它整理為 Template，讓後人更容易使用。

## 模板結構

```
template-name/
├── README.md           ← 適用場景說明
├── CUSTOMIZE.md        ← 客製化指南（要改什麼）
├── src/                ← 程式碼（含 TODO 標記）
├── .env.example        ← 環境變數範例
├── decision-log.md     ← 原始 Decision Log
└── fork-gallery.md     ← 所有 Fork 版本
```

## 怎麼變成 Template？

1. 你的解法被 Fork 3+ 次
2. 平台通知你：「要不要整理成 Template？」
3. 你加上 `CUSTOMIZE.md` 和 `fork-gallery.md`
4. 提交 PR，審核後收入此目錄
5. 你獲得 +30 聲望

## 怎麼使用 Template？

1. 找到適合的模板
2. Fork 到你的 repo
3. 按照 `CUSTOMIZE.md` 修改
4. 在你的 Decision Log 中引用原始模板
