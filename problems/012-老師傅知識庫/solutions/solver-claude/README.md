# #12 老師傅知識庫

## 這是什麼

一個 **Web App**（單一 HTML 檔），讓工廠老師傅用「講的」存經驗，新人用「問的」找答案。

**[▶ 直接打開 index.html 試用](index.html)**

## 兩個模式

| 模式 | 誰用 | 做什麼 | 怎麼用 |
|------|------|--------|--------|
| 🔍 新人提問 | 新進員工 | 打字問問題 → AI 從知識庫找答案 | 輸入問題，按 Enter |
| 🎤 師傅錄入 | 老師傅 | 按住講話 → AI 整理成知識卡片 | 按住大按鈕說話，放開後按「AI 整理」 |

## 系統架構

```
師傅按住講話
    ↓ Web Speech API（瀏覽器內建，免費）
原始語音轉文字
    ↓ Gemini API（免費）
結構化知識卡片（標題/問題/解法/情境/注意事項）
    ↓ 存入 localStorage
知識庫
    ↑ 新人輸入問題
    ↑ Gemini 比對所有卡片，找最相關的回答
```

## 怎麼用

```
1. 打開 index.html
2. 點「師傅錄入」→ 設定 API Key（免費 Gemini）→ 填師傅名字
3. 按住大按鈕講話 → 放開 → 按「AI 整理成知識卡片」→ 確認儲存
4. 切到「新人提問」→ 打字問問題 → 看答案
```

沒有 API Key 也能先體驗：首頁按「載入 10 筆範例資料」看知識卡片長什麼樣。

## 不限產業

同架構換內容就能用：

| 產業 | 師傅 = 誰 | 知識 = 什麼 |
|------|----------|------------|
| 塑膠射出 | 老師傅 | 模具、參數、品質處理 |
| CNC 加工 | 資深技師 | 刀具、進給、材料特性 |
| 餐飲 | 主廚 | 食譜、火候、備料眉角 |
| 汽修 | 老技師 | 故障判斷、維修步驟 |
| 農業 | 老農 | 種植時機、病蟲害判斷 |

## 技術規格

- **單一 HTML 檔**，不需伺服器
- 語音辨識用瀏覽器內建 Web Speech API（需 Chrome）
- 知識卡片存 localStorage
- AI 呼叫走使用者自己的 API Key（sessionStorage，關頁面即清除）
- 手機可用（按住錄音按鈕）
- 內建 10 筆塑膠射出成型的範例知識卡片
