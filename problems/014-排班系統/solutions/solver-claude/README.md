# #14 AI 排班助手

## 這是什麼

一個 Web App，解決餐飲/零售業每週花 4-6 小時手動排班的問題。

**不靠 AI API** — 排班演算法全部在瀏覽器本地跑，不需要網路。

**[▶ 直接打開 index.html 試用](index.html)**

## 競品研究

| 競品 | 問題 |
|------|------|
| [Timefold](https://github.com/TimefoldAI/timefold-solver) (1.6k⭐) | 要 Java runtime，太重 |
| [AutoShiftPlanner](https://betaiotazeta.github.io/AutoShiftPlanner/) | 沒有勞基法、沒有中文 |
| NUEIP、Workstem、Aibou | 商業 SaaS，要月費 |

**我們的定位**：台灣第一個內建勞基法約束、零安裝、免費的排班工具。

## 功能

| Tab | 做什麼 |
|-----|--------|
| 👥 設定 | 門市班別 + 匯入員工可用時段 |
| 📅 排班 | 一鍵自動排班（貪心+約束，本地跑） |
| 🔄 請假換班 | 選人選日 → 建議替代人選 |
| ⚖️ 公平性 | 工時/假日班/班別分配視覺化 |

## 內建勞基法約束

- 每日正常工時 ≤ 8 小時
- 每週正常工時 ≤ 40 小時
- 每 7 天至少 1 天例假 + 1 天休息日
- 連續工作不超過 6 天
- 假日班公平分配（差異 ≤ 1）

## 技術

- 排班演算法：貪心 + 約束滿足（不需要 AI API）
- 公平性排序：最少工時優先 + 最少假日班優先
- 假日自動 ×1.5 人力
- 匯出 CSV（可用 Excel 開）
- 單一 HTML 檔，零安裝，離線可用
- 內建 15 人模擬資料（6 正職 + 9 工讀）
