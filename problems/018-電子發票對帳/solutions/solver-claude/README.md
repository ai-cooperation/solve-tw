# #18 發票對帳 AI 助手

## 這是什麼

一個 **Web App**（單一 HTML 檔），處理中小企業最頭痛的發票作業：

**紙本發票** → 拍照 → AI 辨識所有欄位 → 自動分類科目 → 對帳 → 報表

**[▶ 直接打開 index.html 試用](index.html)**

## 功能

| Tab | 做什麼 | 怎麼用 |
|-----|--------|--------|
| 📸 拍照辨識 | 拍紙本發票 → Gemini Vision 辨識全部欄位 | 拍照/上傳 → 按辨識 |
| 📥 匯入 CSV | 電子發票 CSV 匯入 + 自動分類科目 | 貼上 CSV 或載入 50 筆模擬資料 |
| 🔗 對帳 | 發票 vs 訂單自動比對，標記差異 | 按對帳，看配對結果 |
| 📊 報表 | 各科目支出統計 + 異常偵測 | 按產生報表 |
| ⚙️ 設定 | API Key + 會計科目管理 | 填 Key、自訂科目 |

## 系統架構

```
紙本發票（痛點所在）              電子發票（已結構化）
    ↓ 拍照上傳                        ↓ CSV 匯入
    ↓ Gemini Vision API               ↓ 本地解析
    ↓ JSON 結構化                     ↓
    └──────────┬──────────────────────┘
               ↓
    規則引擎自動分類會計科目（本地，不呼叫 API）
               ↓
    ┌──────────┼──────────┐
    ↓          ↓          ↓
  對帳比對   重複偵測   月報表
  (vs 訂單)  金額異常   科目統計
```

## 怎麼用

```
1. 打開 → 設定 → 填 Gemini API Key
2. 拍照辨識：拍紙本發票 → AI 自動辨識 → 確認存入
   或 匯入 CSV：載入 50 筆模擬資料體驗
3. 對帳：載入訂單 → 一鍵比對 → 看配對結果和異常
4. 報表：看各科目支出、異常標記
```

## 不限產業

會計科目可在設定裡自訂，預設 12 個常用科目。

## 技術規格

- 單一 HTML 檔，不需伺服器
- 紙本辨識用 Gemini Vision API（免費額度）
- CSV 分類用本地規則引擎（不呼叫 API）
- 對帳、報表、異常偵測全部本地計算
- API Key 用 sessionStorage，關頁面即清除
- 內建 50 筆模擬發票 + 模擬訂單
