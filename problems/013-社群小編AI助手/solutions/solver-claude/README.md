# #13 一人小編 AI 助手

## 這是什麼

一個 **Web App**（單一 HTML 檔），讓中小企業老闆 3 分鐘搞定社群貼文。

打開 → 填品牌資料（只做一次）→ 之後每天選功能、按按鈕、複製貼上、發文。

**[▶ 直接打開 index.html 試用](index.html)**

## 功能

| Tab | 做什麼 | 花多久 |
|-----|--------|--------|
| ⚙️ 品牌設定 | 填你的店名、個性、風格 → 儲存 | 一次 5 分鐘 |
| 📅 月排程 | 輸入月份和活動 → AI 排好 30 天 | 每月 2 分鐘 |
| ✏️ 今日文案 | 描述照片 → AI 寫 IG 貼文 + hashtag | 每天 3 分鐘 |
| ⭐ 負評回覆 | 貼上負評 → AI 寫回覆（不卑不亢）| 每則 1 分鐘 |
| 📱 LINE 推播 | 輸入活動 → 3 種風格文案 | 每則 2 分鐘 |

## 怎麼用

```
1. 下載 index.html（或直接打開）
2. 填 API Key（免費的 Google Gemini 就行，aistudio.google.com 申請）
3. 填品牌資料 → 儲存
4. 切到你要的功能 → 填空 → 按按鈕 → 複製結果 → 去發文
```

## 不限行業

品牌設定換掉就好，程式碼不用改：

| 行業 | 品牌個性選 | 主力商品填 |
|------|-----------|-----------|
| 早餐店 | 鄰居阿姨 | 蛋餅、吐司、漢堡 |
| 美髮沙龍 | 好朋友 | 剪髮、染髮、護髮 |
| 水電行 | 專業顧問 | 修繕、安裝、檢測 |
| 牙醫 | 專業顧問 | 洗牙、補牙、矯正 |
| 健身房 | 熱血派 | 團課、私教、體驗 |

## 技術規格

- **單一 HTML 檔**，不需要伺服器，不需要安裝
- 品牌資料存在瀏覽器 localStorage
- AI 呼叫走使用者自己的 API Key（Gemini 免費 / OpenAI）
- 手機優先的 RWD 設計
- 零依賴，離線也能開（AI 功能需要網路）

## 檔案
