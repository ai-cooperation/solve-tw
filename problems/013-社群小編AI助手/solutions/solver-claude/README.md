# #13 一人小編 AI 系統

> **一個人管社群太累？花 30 分鐘設定一次，之後每天 3 分鐘搞定。**

---

## 兩個問題，兩種解法

| 問題 | 解法 | 效果 |
|------|------|------|
| **簡化時間**（每天重複做的事太多）| 月排程 + 文案產生器 → 每天照表操課 | 30 min/天 → 3 min/天 |
| **提升效率**（花的時間沒有累積）| 品牌 DNA 做一次永久沿用 → 越用越快 | 第 1 次 30 min，之後 0 min |

## 怎麼用這個 Repo

### 方法 1：Vibe Coding 建 App（推薦）

把 [vibe-coding-prompt.md](src/vibe-coding-prompt.md) 的內容貼進以下任一工具，它會幫你建一個完整的 Web App：

| 工具 | 網址 | 費用 |
|------|------|------|
| **Replit** | replit.com | 免費 |
| **Bolt** | bolt.new | 免費 |
| **Lovable** | lovable.dev | 免費額度 |

**步驟**：
1. 打開上面任一工具
2. 複製 [vibe-coding-prompt.md](src/vibe-coding-prompt.md) 的全部內容
3. 貼上，按 Enter
4. 等它建好（約 2-3 分鐘）
5. 你會得到一個可以用的 Web App

### 方法 2：手動用 AI 對話

如果不想建 App，也可以直接把 [brand-dna-setup.md](src/brand-dna-setup.md) 貼進 Claude/ChatGPT 對話，用對話方式操作。
