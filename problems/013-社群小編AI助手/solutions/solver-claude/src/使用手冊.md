## 第三步：怎麼用

### 圖解步驟

```
1. 打開上面檔案中的 Prompt（灰色方框的部分）
2. 全選、複製
3. 到 AI 工具的對話框，貼上
4. 把【】裡面的內容改成你自己的
5. 按 Enter 送出
6. AI 回覆後，複製你需要的部分
7. 貼到 IG / Google / LINE
```

### 實際操作示範（以 IG 貼文為例）

**你貼進去的東西：**

```
你是一間台灣連鎖早餐店的社群小編。寫作風格：親切、有溫度......

📷 餐點描述：今天的培根起司漢堡，培根煎到脆脆的
📌 想強調的重點：純粹分享
🎯 今天是：星期五，天氣很好
```

**AI 會回你：**

> 星期五的早餐就是要豐盛一點 ☀️
>
> 培根煎到邊邊脆脆的、起司剛好融化⋯⋯這種組合不需要理由，好吃就是好吃。
>
> 週末快到了，先用一份好吃的漢堡開場吧！
>
> #台灣早餐 #早餐店日常 #培根漢堡 #週五的小確幸 #漢堡控

**你要做的事：複製文案，貼到 IG 發文框。搞定！**

## 常見問題

### 「AI 產出的文案怪怪的怎麼辦？」

直接跟 AI 說你要改什麼。例如：

- 「太長了，縮短到 50 字」
- 「語氣太正式了，要更口語一點」
- 「不要用那個 emoji，換一個」
- 「hashtag 換掉，我們不用英文 tag」

**你可以一直叫它改，不用客氣。**

### 「每次都要貼一大段 Prompt 很麻煩？」

兩個省力方法：

1. **手機備忘錄**：把 4 個 Prompt 存在手機備忘錄，要用的時候複製貼上
2. **AI 的對話記憶**：在 ChatGPT 或 Claude 開一個新的對話，第一次貼完整 Prompt 後，後續在同一個對話裡只需要填新的內容就好

### 「我可以一次叫 AI 幫我做一整週的嗎？」

可以！用 Prompt 3（月曆產生器）先規劃整個月，再用 Prompt 1 逐天產出文案。

建議流程：
```
每月底花 30 分鐘：
用 Prompt 3 產出下個月的行事曆
     ↓
每天早上花 3 分鐘：
照行事曆的主題，用 Prompt 1 產出當天文案
     ↓
遇到負評：
用 Prompt 2 回覆
     ↓
要發 LINE 推播時：
用 Prompt 4 產出文案
```

### 「我不會用 AI，可以叫店裡工讀生幫忙嗎？」

當然可以。把這份手冊給他看就好。
這些 Prompt 設計成「填空題」，會填空就會用。

## 進階技巧（等你熟了再看）

### 讓 AI 學你的風格

如果你覺得 AI 寫的跟你平常的語氣不太像，可以在 Prompt 開頭加上：

```
以下是我之前寫過的 3 則 IG 貼文，請學習我的風格：

1. [貼上你之前寫的第一則]
2. [貼上你之前寫的第二則]
3. [貼上你之前寫的第三則]

請用跟上面類似的風格幫我寫新的貼文。
```

### 一次產出多個版本

在 Prompt 最後加上：`請產出 3 個版本讓我選`，AI 就會給你 3 種不同的文案。

### 存下好用的結果當範本

AI 寫得好的文案，截圖或複製存起來。下次可以跟 AI 說「風格跟上次這則類似」。
