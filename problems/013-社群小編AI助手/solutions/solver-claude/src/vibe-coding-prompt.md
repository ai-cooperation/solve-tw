# Vibe Coding Prompt — 貼進 Replit / Bolt / Lovable 就能建 App

> 複製下面整段，貼進 Replit Agent / Bolt.new / Lovable.dev，等它建完。

---

```
請幫我建一個「一人小編 AI 助手」Web App。
這是給中小企業老闆用的社群內容產生工具，他們不懂技術，介面要極簡。

## 技術規格

- 單頁 Web App（HTML + CSS + JavaScript）
- 不需要後端伺服器，所有 AI 呼叫走使用者自己的 API key
- 支援 OpenAI API 或 Google Gemini API（讓使用者選）
- RWD 手機優先（老闆都用手機）
- 資料存 localStorage（品牌 DNA、歷史產出）
- 中文介面

## 頁面架構

### 首頁（Tab 切換，5 個功能）

**Tab 1：品牌設定（只做一次）**

表單欄位：
- 店名/公司名（文字輸入）
- 做什麼的？幾年了？（文字區域）
- 客人是誰？（文字區域，placeholder：「例：上班族、媽媽帶小孩、學生」）
- 品牌個性（下拉選單：鄰居阿姨/專業顧問/文青風/活潑年輕/你的閨蜜，另有「自訂」選項）
- 講話語氣範例（文字區域，placeholder：「貼 2-3 則你之前寫過的貼文」）
- 絕對不要的風格（文字區域，placeholder：「例：不要太正式、不要用英文」）
- 主力商品/服務（文字區域）
- 社群平台（多選：IG / LINE / FB / Google 商家）

按「儲存」→ 存到 localStorage，顯示「品牌 DNA 已儲存 ✓」
之後所有功能自動帶入品牌 DNA，不用重填。

**Tab 2：月排程**

輸入：
- 月份（年月選擇器）
- 這個月的重要事項（文字區域，placeholder：「例：7/7 七夕、7/15 新品上市、會員日」）

按「產生排程」→ 呼叫 AI，產出表格：

| 日期 | 星期 | 平台 | 主題 | 內容方向 | 類型 |
（類型用 emoji 標籤：🍳產品 💬互動 🎉活動 😄日常 📸回饋）

表格下方有「匯出 CSV」按鈕。
產出的排程也存 localStorage，Tab 3 可以讀取。

**Tab 3：今日文案**

上方自動顯示：「今天是 X/X（星期X），排程主題：____」（從 Tab 2 的排程讀取）
如果沒有排程，就顯示手動輸入欄位。

輸入：
- 今天的主題（自動帶入或手動填）
- 照片描述（文字區域，placeholder：「例：起司蛋餅切開，起司牽絲中」）
- 想強調什麼（文字區域，選填）

按「產生文案」→ 呼叫 AI，產出：
- IG 文案（80-120 字 + 5 hashtag）
- 建議發文時間

下方有「複製文案」按鈕（一鍵複製到剪貼簿）。
另有「再產一版」按鈕（不滿意可以重跑）。

**Tab 4：負評回覆**

輸入：
- 客人評論（文字區域，貼上原文）
- 評分（1-5 星，星星點選）
- 哪家分店（文字輸入，選填）

按「產生回覆」→ 呼叫 AI，產出回覆（80-120 字）

有「複製回覆」按鈕。

**Tab 5：LINE 推播**

輸入：
- 活動主題（文字輸入）
- 活動時間（文字輸入）
- 目標（下拉：促銷來客 / 告知新品 / 通知事項）
- 補充資訊（文字區域，選填）

按「產生文案」→ 產出 3 種風格：
1. 🔥 直接型（標題 + 內文 + CTA）
2. 💛 溫馨型（標題 + 內文 + CTA）
3. 😆 趣味型（標題 + 內文 + CTA）

每種都有「複製」按鈕。

### 設定頁（齒輪圖示）

- AI 服務選擇（下拉：OpenAI / Google Gemini）
- API Key 輸入框（密碼欄位，存 localStorage）
- 「測試連線」按鈕
- 清除所有資料按鈕

## AI 呼叫規則

所有 AI 呼叫的 system prompt 都要帶入品牌 DNA（從 localStorage 讀取）。

system prompt 範本：
```
你是一位台灣中小企業的社群內容專家。
以下是這間店的品牌資料，所有產出都要符合這個風格：

【品牌 DNA 內容從 localStorage 帶入】

寫作規則：
- 語氣要像品牌個性描述的那樣
- 口語、像在講話，不是寫文章
- 不要用「絕對」「必須」「趕快」「錯過可惜」等推銷詞
- 不要用「本店秉持」「竭誠服務」等官腔
- emoji 最多 2 個，要自然
- 中文，繁體
```

## UI 設計要求

- 配色：白底 + 淡黃/橘暖色（早餐店/溫馨感）
- 字體大：最小 16px（老闆娘可能 40-50 歲）
- 按鈕大：最小 48px 高
- Tab 切換在頂部，用圖示 + 文字
- 產出結果用卡片呈現，有明顯的「複製」按鈕
- 載入中顯示「AI 正在寫⋯」動畫
- 手機版單欄排版

## 第一次開啟的引導

如果 localStorage 沒有品牌 DNA 和 API Key：
1. 顯示歡迎頁：「歡迎！先花 5 分鐘做兩件事，之後就輕鬆了」
2. 第一步：設定 API Key（附圖文教學連結）
3. 第二步：填寫品牌 DNA
4. 都完成後才能用其他功能
```

## 套用到其他行業

品牌 DNA 換掉就好，系統架構完全不用改：

| 行業 | 品牌個性改成 | 主力內容改成 |
|------|------------|------------|
| 美髮沙龍 | 閨蜜設計師 | 作品前後對比 |
| 水電行 | 靠譜的大哥 | 施工對比 + 衛教知識 |
| 牙醫診所 | 溫和的醫師 | 口腔衛教 + 診所日常 |
| 健身房 | 熱血教練 | 學員成果 + 運動知識 |
| 花店 | 浪漫的朋友 | 花藝作品 + 花語分享 |
