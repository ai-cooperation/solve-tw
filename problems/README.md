## 題目目錄結構

每個題目在此目錄下有獨立的子目錄：

```
problems/
├── README.md                              ← 你正在看的這個檔案
├── 001-客訴分類系統/
│   ├── README.md                          ← 題目描述
│   ├── data/                              ← 脫敏資料（如有）
│   └── solutions/
│       ├── team-alpha/
│       │   ├── README.md                  ← 解法說明
│       │   ├── DECISION-LOG.md            ← Decision Log（必填）
│       │   ├── src/                       ← 程式碼
│       │   └── REVIEW.md                  ← 評審評語
│       └── solver-bob/
│           └── ...
├── 002-庫存預測/
│   └── ...
```
